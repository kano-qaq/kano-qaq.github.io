MFC原理

Posted on 2023-11-20

MFC类设计

运行时类型识别(RunTime Type Identify)
通过使用结构体记录类型信息，结构体间通过指针记录基类和派生类的关系。
通过virtual关键字实现动态获取运行时类型。
通过函数指针实现类型的实例化

序列化设计
通过全局的构造函数将需要反序列化的类信息插入到链表中

统一消息处理
1.通过修改消息处理函数实现统一消息处理
2.通过虚函数实现获取不同的消息映射表
3.通过保存基类的消息映射表实现完全的多态
4.通过消息处理函数的虚函数，提供用户自行处理的机会

PE01

Posted on 2023-10-17

PE格式

PE格式是用于存储程序创建时所需要的信息。

有以下结构体定义
IMAGE_DOS_HEADER
IMAGE_NT_HEADERS
IMAGE_FILE_HEADER
IMAGE_OPTIONAL_HEADER
IMAGE_SECTION_HEADER

添加section
a:
1.在Section Headers添加一个合理的IMAGE_SECTION_HEADER信息。
2.在文件中添加相应的数据
3.修改IMAGE_FILE_HEADER中的NumbersOfSections字段
4.修改IMAGE_OPTIONAL_HEADER中的SizeOfImage字段
b:
1.修改最后一个Section Headers中的文件大小
2.在文件中添加相应数据
c:(等价于c语言中的未初始化section)
1.在Section Headers添加一个Virtual Size位为0的合理的IMAGE_SECTION_HEADER信息。
2.不需要在文件中添加数据
d:
1.增加最后一个Section Headers中的Virtual Size
