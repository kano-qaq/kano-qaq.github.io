套接字设置TimeOut
1.创建一个fd_set结构体fd

5.select(0,&read_fd,&write_fd,&except_fd,&tv)为传入的fd集合设置TimeOut
6.使用宏FD_ISSET(套接字,&fd)判断是否超时

Posted on 2023-07-26 Edited on 2023-08-31

TCP协议
1.三次握手建立连接,四次挥手断开连接
三次握手:
Client -> Server SYN=m
Server -> Client SYN=n,ACK=m+1
Client -> Server SYN=m+1,ACK=n+1
四次挥手:
Client -> Server FIN=j,ACK=k
Server -> Client ACK=j+1 //此处断开客户端到服务端的连接，客户端仍可接收服务端的数据
Server -> Client FIN=k
Client -> Server ACK=k+1 //此处断开服务端到客户端的连接

2.TCP通过窗口进行流量控制和提高发送的效率
3.TCP在发送时会出现粘包的现象

网络通信

1.创建套接字
Socket=socket(AF_INET(IPv4),SOCK_DGRAM(报文),IPPROTO_UDP(UDP协议))
2.初始化sockaddr结构体,其中地址和端口都需要转为大端序
3.绑定端口
bind(Socket,&sockaddr_self,sizeof(sockaddr))
4.发送/接收数据
sendto(Socket,Buf,sizeof(Buf),0,sockaddr_to,sizeof(sockaddr_to))
recvfrom(Socket,Buf,sizeof(Buf),0,sockaddr_from,sizeof(sockaddr_from))
recvfrom函数会阻塞线程。

在windows中需要调用WSAStartup()函数才可以创建套接字,退出时调用WSACleanup()

Posted on 2023-07-19 Edited on 2023-08-31

服务
1.程序的特点:长期在后台运行的程序，无需UI，为所有进程提供特定的功能
2.服务可以开机后自启动
3.服务由服务管理器(SCM)管理，可以通过SCM来安装、开始、停止和删除服务
4.安装的服务会在注册表的HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services目录下添加相应的表项
5.服务的入口函数为ServiceMain(),可以通过调用StartServiceCtrlDispather()把main转为ServiceMain.
6.服务的每次状态转换都存在pending状态，服务只在运行状态工作.

服务程序基本框架
安装:
1.获取服务程序路径
2.打开SCM管理器
3.创建服务
开始\停止:
1.打开SCM管理器
2.打开指定服务
3.查询服务状态
4.若服务未运行，则运行服务\若服务未停止，则停止服务

Posted on 2023-07-18 Edited on 2023-08-31

消息钩子

局部钩子
1.SetWindowsHookEx(需要监听的消息,消息处理函数,NULL,0)
2.在消息处理函数中需要调用CallNextHookEx(),钩子是个链表,且后hook的先调用
3.不使用钩子时需要卸载

全局钩子
1.SetWindowsHookEx(需要监听的消息,消息处理函数,动态库的基址,0)
2.在消息处理函数中需要调用CallNextHookEx()
3.不使用钩子时需要卸载
4.Hook时动态库的位数最好和进程一样

全局钩子本质:操作系统往所有进程内注入给定的动态库
Ctrl+d可以设置窗口顺序

Posted on 2023-07-15 Edited on 2023-08-31

线程同步

硬件同步:
1.原子锁,将对变量的操作从硬件上处理为不可分割。如InterlockedIncrement(&变量地址)

软件同步
1.临界区,通过设置临界区保证共享变量一次只能有一个线程访问
InitializeCriticalSection(&g_cs);//初始化临界区
EnterCriticalSection(&g_cs);//进入临界区
…//访问资源
LeaveCriticalSection(&g_cs);//离开临界区
DeleteCriticalSection(&g_cs);//释放临界区

进程同步
1.互斥体(没有通知)
g_hMutex = CreateMutex(NULL, FALSE, NULL);//创建互斥体
WaitForSingleObject(g_hMutex, INFINITE);//等待互斥体
ReleaseMutex(g_hMutex);//释放互斥体

2.信号量(可一个创建多个信号量用于通知)
g_hSem = CreateSemaphore(NULL, 1, 1, NULL);//创建一个信号量
WaitForSingleObject(g_hSem, INFINITE);//等待信号量
ReleaseSemaphore(g_hSem, 1, NULL);//释放一个信号量

3.事件对象(在创建时分为手动和自动,手动用于通知,自动用于同步)
g_hEvent = CreateEvent(NULL, FALSE, TRUE, NULL);//创建一个自动的事件对象
WaitForSingleObject(g_hEvent, INFINITE);//等待事件对象
SetEvent(g_hEvent);//设置事件对象有效
