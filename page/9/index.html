CR-Day17

Posted on 2023-04-07

note

结构体空间计算：
设编译选项对齐值为Zp
设成员的数据类型为member type
设成员偏移量member offset
必须满足：
member offset % min(Zp,sizeof(member type)) == 0

设结构体变量的对齐值为StructAlig
StructAlig = max(sizeof(member1 type),sizeof(member2 type) …)
设结构体变量的内存空间占用为StructSize
必须满足：
StructSize % min(Zp,StructAlig) == 0

对齐值操作
1.#pragma pack(push) 保存当前对其值
2.#pragma pack(2) 修改对齐值为2
3.#pragma pack(pop) 恢复保存的对齐值

结构体操作等价式
Struct tagTest t=…;
t.member or (&t)->member address is:
*(member type *)((int)&t+member offset)

Offsetof(struct type,b)=b’s offset in struct type
