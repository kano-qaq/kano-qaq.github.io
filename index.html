1.link -lib /list 目标.lib 显示目标库中的所有obj

3.pcf object1.obj 将.obj文件转为.pat文件

除法只有除数不为变量时可以优化。

无符号作被除数:
除数为2^n时:
除数大于0:shr 被除数,n
除数小于0:div 除数

除数不为2^n时:
除数的10进制表达式=floor(2^(32+右移位数)/magic_number)
将除数的10进制表达式转为16进制判断正负
(乘减移加移的真实magic_number = 1 0000 0000h + 反汇编显示的magic_number)

vc6中:
unsigned int / -5 和 unsigned int / 4294967291 汇编代码一样
vs2019中:
表达式unsigned int / 4294967291的4294967291会默认转为long long类型，导致引入大数乘法。
unsigned int / -5 和unsigned int / 4294967291i32 的汇编代码一样

MFC原理

Posted on 2023-11-20

MFC类设计

运行时类型识别(RunTime Type Identify)
通过使用结构体记录类型信息，结构体间通过指针记录基类和派生类的关系。
通过virtual关键字实现动态获取运行时类型。
通过函数指针实现类型的实例化

序列化设计
通过全局的构造函数将需要反序列化的类信息插入到链表中

统一消息处理
1.通过修改消息处理函数实现统一消息处理
2.通过虚函数实现获取不同的消息映射表
3.通过保存基类的消息映射表实现完全的多态
4.通过消息处理函数的虚函数，提供用户自行处理的机会
