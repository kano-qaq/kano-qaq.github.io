note

术语：
数据(Data)
元素(Element)
数据元素(Data Element) ：数据的基本单位
字段（Field)
数据项(Item)：数据的不可分割最小单位
数据对象(Data Object)：性质/类型相同的数据元素的集合
记录(Record)
节点(Node)
层(Layer)
父亲(Parent)
孩子(Child)

常见数据结构：
1.线性结构前后关系
2.树结构
3.图结构

算法的特征：有限性，确定性，可行性，输入，输出
算法的要求：正确性，可读性，健壮性，高效与低存储量

时间增长率 => 时间复杂度
空间增长率 => 空间复杂度

O(1) => 常量阶
O(logN) => 对数阶
O(N) => 线性阶
O(N^2) => 平方阶
O(N^3) => 立方阶

最好时间复杂度:O(N^2)
最差时间复杂度:O(N^2)

最好时间复杂度:O(N)
最差时间复杂度:O(N^2)

平均时间复杂度:最好时间复杂度占比最好时间复杂度+最差时间复杂度占比最差时间复杂度

总频度：f(N）
时间复杂度：T(N)
